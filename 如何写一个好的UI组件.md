# 封装点 
- DOM结构
    - CSS还做不到不依赖DOM就能任意定制样式, 因此DOM结构需要被封装
    - 暴露Template可以最大程度定制DOM结构
    - 暴露JS API完成复杂的DOM操作, 或者用于保障原子操作
- 样式
    - 提供基本样式, 方便做快速原型开发
    - 如果能暴露稳定的CSS Selector, 就能最大程度定制样式
- 交互
    - 提供enable/disable的能力
        - 交互需求很容易变化, 当需求变化时至少还能重用组件的其他功能
        - disable时清除所有的事件, 这样可以减少内存消耗
    - 使用FSM完成逻辑的设计    
- 事件
    - 使用DOM4事件类型可以避免自己设计事件
